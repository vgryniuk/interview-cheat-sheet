## Java 5
### Variable Arguments (Varargs)
Метод, що може використовувати змінну кількість аргументів одного типу.
```
static void m(int... a) {
    System.out.println("Number of arguments: " + a.length);
    for (int i : a) {
        System.out.println(i + " ");
        }
}
```
